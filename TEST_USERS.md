# 🧪 Utilisateurs de Test - Quran School SaaS

Voici la liste des comptes pré-configurés pour tester l'application.
Le mot de passe est généralement le nom du rôle suivi de `@123`.

## 🏫 Accès Application
- **Web** : [http://localhost:3000](http://localhost:3000)
- **API** : [http://localhost:5000/swagger](http://localhost:5000/swagger)

---

## 👤 Super Administrateur (Plateforme)
Accès complet à toutes les écoles et configuration système. S'affiche comme "Super Administrateur" dans le tableau de bord.
- **Email** : `superadmin@quranschool.com`
- **Mot de passe** : `Admin@123`

## 👔 Administrateur (Directeur d'école)
Gestion d'une école spécifique (Al-Noor). S'affiche comme "Administrateur" dans le tableau de bord.
- **Email** : `admin@alnoor-quran.fr`
- **Mot de passe** : `Admin@123`

## 👨‍🏫 Teacher (Professeur)
Accès aux groupes, présence, saisie des notes et progrès.
- **Email** : `teacher@alnoor-quran.fr`
- **Mot de passe** : `Teacher@123`

## 👨‍👩‍👧 Parent (Responsable)
Suivi des enfants, paiements et communication.
- **Email** : `parent@example.com`
- **Mot de passe** : `Parent@123`

---

> **Note** : Ces données sont réinitialisées si vous supprimez le fichier de base de données `quranschool.db`.
